Show information about a medium.

Check if the configuration is correct.

## Example

```powershell
VBoxManage showmediuminfo disk "C:\Users\xanet\VirtualBox VMs\Linux\xanet-zaldua\eraikin-nagusia\zaldua1zerb1\zaldua1zerb1-disk.vdi"
```

## Command Options

```powershell
VBoxManage showmediuminfo [disk | dvd | floppy] <uuid | filename>
```
